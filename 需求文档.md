# 取名工具 - 需求文档

## 📋 项目概述
一个基于 AI 的取名工具，用户点击「取名」按钮，AI 实时生成一批名字并以词云形式展示，点击名字可查看详细寓意。用户可收藏心仪的名字到「我的仓库」并打分，也可将不喜欢的名字或风格加入「抛弃池」，后续生成自动规避，逐渐优化生成效果。

## 🎯 核心功能

### 1. 取名对象
- **给人取名**（宝宝取名、艺名等）

### 2. 名字数据来源
- **AI 实时生成**：每次点击「取名」，调用专门取名的大模型 API 生成一批名字和寓意
- **每次至少生成 50 个名字**
- **去重机制**：已生成过的名字不再重复出现（基于本地存储记录）
- **抛弃池规避**：被用户抛弃的名字及风格，后续生成时自动过滤

### 3. 用户输入（可选）
- **姓氏**：可选输入，AI 结合姓氏生成名字
- **风格偏好**：可选输入（如文雅、大气、现代、古典等），AI 按风格生成
- **不输入**：AI 随机自由发挥

### 4. 词云展示
- 展示内容：**名字本身**（如"安澜"、"致远"）
- 字体大小：按寓意评分或随机
- 交互：点击名字 → 弹出详细寓意
- 词云中提供「收藏」和「抛弃」快捷操作

### 5. 寓意详情（弹窗展示）
点击名字后展示：
- **名字**（如"安澜"）
- **单字解析**（安 = 平安安宁；澜 = 波澜壮阔）
- **整体寓意**（如"波澜不惊，安然自若"）
- **适合性别**（男孩/女孩/通用）
- **风格标签**（如文雅、大气）
- **诗词出处/典故**（如有）
- **评分**
- **操作按钮**：收藏 / 抛弃

### 6. 我的仓库（收藏池）
- **存放位置**：浏览器本地存储（localStorage），无需登录
- **展示方式**：列表/卡片形式展示所有收藏的名字
- **每条记录包含**：名字、寓意摘要、收藏日期、个人评分
- **操作**：
  - 查看详情（再次查看完整寓意）
  - 自行打分（1-5 星或 1-10 分）
  - 删除/移除

### 7. 抛弃池
- **存放位置**：浏览器本地存储（localStorage）
- **存放内容**：
  - 用户明确抛弃的**名字**（后续不再生成）
  - 用户明确抛弃的**风格**（如不喜欢"古典风"，后续规避）
- **展示方式**：可查看抛弃列表，支持恢复（移回可用池）
- **作用**：每次生成时，AI prompt 中携带抛弃池信息，自动规避，**逐渐优化生成效果**

## 🛠 技术方案
- **形态**：纯前端 Web 页面（HTML + CSS + JS）
- **存储**：浏览器 localStorage（我的仓库 + 抛弃池数据持久化）
- **部署**：先本地运行，后续再考虑部署到公网
- **AI 接口**：调用取名大模型 API

## 🎨 页面流程
```
首页（输入姓氏/风格，可选）
    ↓ 点击「取名」
调用 AI 生成至少 50 个名字（已出现过的 + 抛弃池中的自动过滤）
    ↓
展示词云（名字随机分布，大小不一）
    ↓ 点击某个名字
弹出详情弹窗（单字解析、寓意、出处等）
    ├─ 点击「收藏」→ 加入我的仓库
    └─ 点击「抛弃」→ 加入抛弃池（后续不再出现）
    ↓
「我的仓库」页面：查看收藏、打分、排序、删除
「抛弃池」页面：查看抛弃记录、恢复名字/风格
```

## 📝 备注
- 姓氏和风格为非必填项
- 词云中的名字字体大小可反映寓意评分
- 每次生成至少 50 个名字
- 已生成过的名字不再重复出现
- 抛弃池中的名字和风格后续生成自动规避，逐步优化推荐效果
- 所有数据存储在浏览器本地，清除浏览器数据会丢失
